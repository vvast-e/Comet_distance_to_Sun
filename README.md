# Разработка интернет-приложений (РИП) 2025
## Расчет расстояния комет до Солнца
Разработанное приложение позволит астрономам создавать и формировать заявки на расчёты расстояния комет до Солнца, а главным астрономам завершать или отклонять данные заявки.
